## ✍️ Commit Convention
어떤 알고리즘을 공부했는지, 스스로 복습이 더 필요한 상태인지 직관적으로 파악할 수 있도록 학습 히스토리를 커밋 메시지에 기록합니다.

문제풀이 추가: [알고리즘 유형 및 문제 수 / 추가 학습 내용 및 복습 여부]
예시:
문제풀이 추가: 이진 탐색 복습1
문제풀이 추가: DP 1문제, 그리디 1차완료(2문제 복습필요)
문제풀이 추가: DFS BFS공부(이전 문제 다시풀기), 최대공약수 최소공배수 관련 유클리드호제법 공부
